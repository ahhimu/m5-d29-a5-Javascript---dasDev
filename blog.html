Question-3: What is event delegation in the DOM?

event delegation: It's Javascript Technique. wheere we use an event on parent elements to handele there child elements. instead of adding an event listener to each child element individually, we place an event on a parent element to manage the behavior of all its children

Question-4: What is event bubbling in the DOM?

event bubbling : It's a Javascript Technique,where we can start from a child element and propagate the event through its parent, then to its parent's parent, and so on, all the way up to the root element in the document.
